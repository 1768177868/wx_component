## 微信开放平台 公众号第三平台类库

wx_component 微信公众号第三平台类库，PHP版本。

wx_component 是根据[微信官方开放平台](https://open.weixin.qq.com/)提供的文档开发的PHP类库，
第三方平台使用此类库，可以在部署代码和代公众号实现业务时，更加容易、方便上手。

## 下载

你可以 clone 这个仓库，自行下载使用。
